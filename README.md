Легкий уровень
В заданной строке определить, совпадает ли количество круглых открывающихся и круглых закрывающихся скобок.

Средний уровень
В заданной строке после каждого символа вставить число, соответсвующее коду этого елемента

Сложный уровень
Заданы две строки. Построить новую строку, состоящию из символов, которые входят в первую строку, но не входят во вторую.
